# Repositorio para testes de comandos do GitFlow 

### Git init: iniciar repositorio git
### Git checkout: muda para a branch
### Git checkout -b: cria uma nova brach  
### Git clone: clona um repositório
### Git add . : Adiciona todos os arquivos para o commit
### Git merge "MAIN": faz um merge do repositorio escolhido para o atual
### Git commit -m: comenta um commit
### Git cherry-pick : Pega um commit do código
### Git commit --amend: edita o último commit
### Git branch: Lista as Branches
### Git branch nome: Cria uma branch
### Git checkout -b mone: Cria e muda para branch
### Git merge branch: junta outra branch na atual
### Git pull: atualiza o repositório local
### Git push: envia commits para o remoto
### Git fetch: baixa mudanças sem aplicar
